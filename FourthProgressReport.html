Learning To Convert Objects into Strings Using JSON.stringify

This is useful and is very easy. If we have an object and we need to convert it to a string we would do the following. If for the object we have a car, let's say for example a car. The make is Honda, the model is Civic and the year is 2016. It would look something like this:

var obj = { make: "Honda", model: "Civic", year: "2016"};

This by itself will not work and it will need to be converted. We would do that by using JSON.stringfy() like so:

var myTest = JSON.stringify(obj);

There we have it! The object has been converted into a string and is now ready to be sent to the server.
